2.1 Factor seguridad

seguridad_model <- '
subjetiva =~ seguridad_perc + seguridad_sat

'

seguridad_cfa <- cfa(model = seguridad_model,
                     data = elsoc_seg,
                     estimator = "DWLS",

2.3 Factor redes sociales

A partir del EFA y la idea de minimalizar la propuesta, se decide plantear un modelo con los mismos dos factores ya existentes: confianza interpersonal y comportamiento prosocial. La diferencia es que se reduce la cantidad de indicadores de 10 a 4, dos pertenecientes a cada factor.

redes_model <- '
confianza_inter =~ confianza_gen + altruismo_gen
prosocial =~ reunion_pub + voluntariado
'
redes_cfa <- cfa(model = redes_model,
                     data = elsoc_redes,
                     estimator = "DWLS",
                     ordered = TRUE,
                     std.lv = FALSE)
